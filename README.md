# MenuBarColor
MenuBarColor - OSX Application. (Swift)

# 概要

* Menubar常駐アプリ
* アイコンをクリックするとFlat/Materialのカラーパレットが表示
* パレットをクリックすると、クリップボードに色がコピーされる
